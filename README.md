# Disease Prediction ML Project

Predict diseases based on symptoms using RandomForest and DecisionTree models.

## Dataset  
Kaggle dataset,Symptom data with severity and precautions from multiple CSV files.

## Results  
- Both models achieved 100% accuracy.  
- Includes confusion matrix and feature importance visuals.

## Files  
- `FA2_Healthcare_Path_Recommendation.ipynb` — Code and analysis  
- `evaluation_metrics.csv` — Accuracy scores  
- `refinement_results.csv` — Model comparison  
- `visuals/` — Visualization images
